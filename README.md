# EncryptedFile

EncryptedFile er et projekt skrevet i blazor, som er til kryptereing af filupload og download af fil, hvor krypteringsnøgler sikkert håndteres eksempelvis gennem Azure Key Vault.


### Forudsætninger

- .NET 6.0 SDK eller nyere
- Azure Key Vault opsat og konfigureret i appsettings.json
- Microsoft SQL Server database konfigureret i appsettings.json

### Installation

1. Klon repo:

git clone https://github.com/yonesswaidan/EncryptedFile.git

cd EncryptedFile

Kør projektet:

dotnet run
